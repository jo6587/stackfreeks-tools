# CSS Filter Previewer

**Live URL:** https://tools.stackfreeks.com/tools/css-filter-previewer/

A real-time CSS `filter:` property previewer and code generator. Adjust sliders, see the result instantly on a sample image, then copy the generated CSS with one click.

## Features

- 9 filter sliders: `blur`, `brightness`, `contrast`, `saturate`, `grayscale`, `hue-rotate`, `sepia`, `invert`, `opacity`
- Live preview on an inline SVG landscape scene
- Generates clean CSS — only non-default values are included
- One-click copy of the `filter:` declaration
- Reset button to restore all defaults
- English / Korean (EN/KO) language toggle
- Fully responsive — stacks vertically on mobile

## Tech

- Pure HTML/CSS/JS — no build step, no dependencies
- Inline SVG for the preview image (colorful landscape with sky, sun, clouds, trees)
- Runs entirely in the browser

## Usage

1. Move any slider to adjust the filter value
2. The SVG preview updates in real time
3. Check the Generated CSS panel for the resulting declaration
4. Click **Copy CSS** and paste into your project
