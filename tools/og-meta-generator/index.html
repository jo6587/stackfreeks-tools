FAQ
What are Open Graph meta tags?
Open Graph (OG) meta tags are HTML snippets placed in a page's <head> that control how the page appears when shared on social media platforms like Facebook, LinkedIn, and Slack. They define the title, description, and image shown in link preview cards.
Open Graph 메타 태그란 무엇인가요?
Open Graph(OG) 메타 태그는 페이지의 <head>에 삽입하는 HTML 코드로, 페이스북·링크드인·슬랙 등에서 링크를 공유할 때 보이는 제목, 설명, 이미지를 제어합니다.
What size should my OG image be?
The recommended OG image size is 1200 × 630 pixels (roughly 1.91:1 ratio). This works well across Facebook, Twitter (summary_large_image), and LinkedIn. For Twitter's "summary" card type, a 1:1 square image (at least 144 × 144 px) is used instead.
OG 이미지 권장 크기는 얼마인가요?
권장 OG 이미지 크기는 1200 × 630 픽셀(약 1.91:1 비율)입니다. 페이스북, 트위터(summary_large_image), 링크드인 모두에서 잘 작동합니다. 트위터 "summary" 카드 유형은 1:1 정사각형 이미지(최소 144 × 144 px)가 사용됩니다.
Why isn't my OG image showing on Facebook?
Facebook caches OG data aggressively. After updating your meta tags, use the Facebook Sharing Debugger to force a cache refresh. Also ensure your image URL is publicly accessible and the server returns the correct Content-Type header.
페이스북에서 OG 이미지가 표시되지 않는 이유는?
페이스북은 OG 데이터를 강하게 캐시합니다. 메타 태그를 업데이트한 후 Facebook Sharing Debugger를 사용해 캐시를 강제 갱신하세요. 이미지 URL이 공개 접근 가능하고 서버가 올바른 Content-Type 헤더를 반환하는지도 확인하세요.
What is a Twitter Card?
Twitter Cards are Twitter's equivalent of OG tags — they control how your link looks when shared in a tweet. The most popular type is "summary_large_image," which shows a large image with the title and description below. You must add Twitter Card meta tags separately from OG tags.
Twitter Card란 무엇인가요?
Twitter Card는 트위터에서 링크를 공유할 때 어떻게 보일지 제어하는 메타 태그입니다. 가장 많이 쓰이는 유형은 큰 이미지와 함께 제목·설명을 표시하는 "summary_large_image"입니다. OG 태그와 별도로 Twitter Card 메타 태그를 추가해야 합니다.
Do meta tags affect SEO?
The <title> and <meta name="description"> tags directly affect how your page appears in Google search results and can influence click-through rates. OG and Twitter Card tags don't directly influence rankings, but better social previews can drive more traffic and indirect SEO signals.
메타 태그가 SEO에 영향을 미치나요?
<title>과 <meta name="description"> 태그는 구글 검색 결과 표시에 직접 영향을 미치며 클릭률(CTR)에도 영향을 줍니다. OG와 Twitter Card 태그는 직접적인 랭킹 요소는 아니지만, 더 나은 소셜 미리 보기는 트래픽을 늘려 간접적인 SEO 신호로 작용합니다.
