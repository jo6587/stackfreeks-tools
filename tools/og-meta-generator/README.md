# Open Graph Meta Tag Generator

> Live: https://tools.stackfreeks.com/tools/og-meta-generator/

Generate Open Graph and Twitter Card meta tags with live previews for Google, Facebook, and Twitter. Free, no account required.

## Features
- Live preview: Google SERP, Facebook card, Twitter card
- All OG properties: title, description, URL, image, site name, type, locale
- Twitter Card types: summary, summary_large_image
- Copy-ready HTML output
- No data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
