# VPS Migration Checklist

> Live: https://tools.stackfreeks.com/tools/vps-migration-checklist/

Interactive checklist generator for migrating a Linux VPS. Select your stack and get a tailored step-by-step plan.

## Features

- Service-aware checklist (Nginx, MySQL, Docker, WordPress, and more)
- Interactive checkboxes with progress tracking
- Copy checklist as plain text
- EN/KO bilingual

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
