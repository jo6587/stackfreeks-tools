# Dockerfile Generator

> Live: https://tools.stackfreeks.com/tools/dockerfile-generator/

Generate a production-ready Dockerfile for Node.js, Python, Go, PHP, Ruby, or Java in seconds — with multi-stage build, non-root user, HEALTHCHECK, and .dockerignore.

## Features

- Supports Node.js, Python, Go, PHP, Ruby, Java (Maven), and Static (nginx)
- Multi-stage build option for slim production images
- Non-root user for container security best practice
- Optional HEALTHCHECK instruction
- Auto-generated .dockerignore per runtime
- Syntax-highlighted output with one-click copy

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
