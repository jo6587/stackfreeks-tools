# Firewall Rule Builder

> Live: https://tools.stackfreeks.com/tools/firewall-rule-builder/

Visual generator for iptables and ufw firewall rules. No syntax memorization needed.

## Features

- Supports both iptables and ufw syntax
- Generate rules for inbound, outbound, and forward traffic
- Build multi-rule scripts with one click
- Copy-ready commands

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
