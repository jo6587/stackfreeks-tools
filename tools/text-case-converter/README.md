# Text Case Converter

> Live: https://tools.stackfreeks.com/tools/text-case-converter/

Convert text between UPPERCASE, lowercase, Title Case, camelCase, snake_case, kebab-case, and more.

## Features
- 10 conversion types
- Live word/character count
- Copy output, swap input/output
- No account, no data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
