# Lorem Ipsum Generator

A free, browser-based lorem ipsum placeholder text generator built for [StackFreeks Tools](https://tools.stackfreeks.com).

**Live:** https://tools.stackfreeks.com/lorem-ipsum/

## Features

- Generate placeholder text as **words**, **sentences**, or **paragraphs**
- Configurable count (up to 100 paragraphs, 500 sentences, or 2,000 words)
- Optional classic "Lorem ipsum dolor sit amet…" opening
- Live **word count** and **character count** stats
- One-click **copy to clipboard**
- Bilingual UI — **English / Korean** toggle
- Fully static — no server, no dependencies, no data sent anywhere

## Tech

- Pure HTML / CSS / JavaScript (no build step)
- Shared i18n system via `/shared/i18n.js`
- Classic lorem ipsum corpus (200+ words)

## Word bank

Sourced from the traditional lorem ipsum corpus originally derived from Cicero's *de Finibus Bonorum et Malorum* (45 BC), plus extended pseudo-Latin vocabulary commonly used in the design industry.
