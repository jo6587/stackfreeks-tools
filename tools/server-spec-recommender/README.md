# Server Spec Recommender

> Live: https://tools.stackfreeks.com/tools/server-spec-recommender/

Find the right VPS plan for your workload. Enter your use case and traffic scale to get instant RAM, CPU, and storage recommendations.

## Features

- 8 workload presets: WordPress, Node.js API, PostgreSQL, Next.js, Docker, CI/CD, Game Server, Static/Nginx
- 4 traffic scale tiers: Personal, Small, Medium, High Traffic
- Optional extras: database on same server, Redis, multiple apps
- Instant recommendation with matching Vultr plan and one-tier-up "comfortable" option
- EN/KO bilingual

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
