# curl Command Generator

> Live: https://tools.stackfreeks.com/tools/curl-command-generator/

Build ready-to-run curl commands — pick method, add headers, auth, and body without memorising flags.

## Features

- HTTP methods: GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
- Authentication: Bearer Token, Basic Auth, API Key header
- Custom headers with add/remove rows
- Request body with Content-Type selector (JSON, form, plain text)
- Options: follow redirects, verbose, silent, insecure, compressed, timeout
- Live command preview with one-click copy
- EN/KO bilingual (i18n)

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
