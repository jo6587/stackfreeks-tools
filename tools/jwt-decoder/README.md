# JWT Decoder

> Live: https://tools.stackfreeks.com/tools/jwt-decoder/

Decode and inspect JSON Web Tokens in your browser. View header, payload, expiry status, and well-known claims.

## Features
- Decode header and payload (base64url)
- Expiry status with countdown
- Well-known claims table (iss, sub, aud, exp, iat)
- Signature not verified (no secret key)
- Nothing sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
