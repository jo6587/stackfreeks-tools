# Linux Swap Size Calculator

> Live: https://tools.stackfreeks.com/tools/linux-swap-calculator/

Calculate the recommended Linux swap size for any VPS or server — with ready-to-paste terminal commands.

## Features

- Instant swap recommendation based on RAM size (512 MB → 256 GB)
- Use-case presets: General VPS, Database, Dev Machine, Desktop
- Hibernation mode (swap ≥ RAM + √RAM)
- Swap file vs. swap partition toggle
- Per-use-case swappiness recommendation (1–60)
- One-click copy of complete Linux commands (fallocate, mkswap, swapon, fstab, sysctl)
- EN / KO bilingual UI

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
