# SSH Config Generator

> Live: https://tools.stackfreeks.com/tools/ssh-config-generator/

Generate a ready-to-paste `~/.ssh/config` file by filling in a simple form — no manual SSH syntax lookup needed.

## Features

- Add multiple SSH host entries via form
- Supports Host, HostName, User, Port, IdentityFile, ServerAliveInterval, ProxyJump, ForwardAgent, StrictHostKeyChecking
- Live config preview with copy and download buttons
- EN/KO bilingual (i18n)

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
