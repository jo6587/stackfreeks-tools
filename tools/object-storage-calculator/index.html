Frequently Asked Questions

Egress (data transfer out) is the fee charged when users download files from your storage bucket to the internet. AWS S3 charges $0.09/GB egress — on a 1TB/month download workload that alone adds up to $92/mo. Cloudflare R2 and Wasabi offer zero egress fees, making them dramatically cheaper for read-heavy workloads.
Yes — R2 egress to the public internet is always free. You pay only for storage ($0.015/GB/month, first 10 GB free) and API requests (Class A writes at $4.50/million, Class B reads at $0.36/million). For assets served through Cloudflare CDN, R2 is one of the most cost-efficient choices available.
Vultr Object Storage shines when you already run workloads on Vultr VPS instances. Internal traffic between Vultr services avoids cross-provider egress entirely. At $0.01/GB storage and $0.01/GB egress with an S3-compatible API, it integrates with any existing S3 SDK without code changes.
Almost always for storage-heavy use cases. B2 charges $0.006/GB — roughly 4× cheaper than S3's $0.023/GB. It also includes free egress equal to 3× your monthly stored data. Paired with Cloudflare CDN (free via the Bandwidth Alliance), B2 effectively becomes zero-egress as well.
Wasabi's egress is free as long as your total data downloaded per month does not exceed your total stored data. If you download more than you store, the excess is billed at $0.04/GB. For typical storage-heavy workloads (backups, archives, media), Wasabi's $0.0059/GB storage with no egress is extremely competitive.

자주 묻는 질문

이그레스(데이터 전송 아웃)는 사용자가 버킷에서 인터넷으로 파일을 다운로드할 때 발생하는 비용입니다. AWS S3는 $0.09/GB를 청구하며, 1TB/월 다운로드 시 이그레스만 $92가 됩니다. Cloudflare R2와 Wasabi는 이그레스가 무료이므로 읽기가 많은 워크로드에서 훨씬 저렴합니다.
네, R2의 공개 인터넷 이그레스는 항상 무료입니다. 스토리지($0.015/GB/월, 처음 10GB 무료)와 API 요청(쓰기 $4.50/백만, 읽기 $0.36/백만)에만 비용이 발생합니다. Cloudflare CDN을 통해 파일을 제공하면 가장 비용 효율적인 선택 중 하나입니다.
Vultr VPS 인스턴스에서 이미 워크로드를 실행 중이라면 Vultr Object Storage가 최적입니다. Vultr 서비스 간 내부 트래픽은 크로스 프로바이더 이그레스가 없으며, S3 호환 API로 기존 SDK를 코드 수정 없이 그대로 사용할 수 있습니다.
스토리지 집약적 워크로드에서는 거의 항상 저렴합니다. B2는 $0.006/GB로 S3의 $0.023/GB보다 약 4배 저렴하며, 월간 저장 용량의 3배를 무료 이그레스로 제공합니다. Cloudflare CDN과 결합하면(Bandwidth Alliance 무료) 사실상 이그레스 비용 없이 사용할 수 있습니다.
Wasabi의 이그레스는 월간 다운로드 총량이 저장 데이터를 초과하지 않는 경우 무료입니다. 초과분은 $0.04/GB로 청구됩니다. 백업·아카이브·미디어 등 스토리지 집약적 워크로드에서는 $0.0059/GB의 낮은 스토리지 요금과 무료 이그레스가 매우 경쟁력 있습니다.
