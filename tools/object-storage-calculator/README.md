# Object Storage Cost Calculator

> Live: https://tools.stackfreeks.com/tools/object-storage-calculator/

Compare monthly object storage costs across AWS S3, Backblaze B2, Cloudflare R2, Vultr Object Storage, Wasabi, and DigitalOcean Spaces — based on your actual storage and bandwidth usage.

## Features

- Input storage (GB) and monthly egress (GB) via sliders or text fields
- Instant cost comparison sorted cheapest to most expensive
- Handles provider-specific free tiers (Backblaze 3× free egress, R2 10 GB free storage, DO Spaces 250 GB + 1 TB flat fee)
- Highlights Vultr Object Storage with direct affiliate CTA
- EN/KO bilingual with i18n toggle
- FAQ and how-to sections for SEO long-tail coverage

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
