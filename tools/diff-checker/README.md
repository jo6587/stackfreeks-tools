# Diff Checker

> Live: https://tools.stackfreeks.com/tools/diff-checker/

Compare two texts side by side and highlight added, removed, and unchanged lines. Free, no account, no file size limit.

## Features
- LCS-based line diff algorithm
- Side-by-side and unified diff views
- Line numbers
- Added/removed/unchanged stats
- Copy diff output
- No data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
