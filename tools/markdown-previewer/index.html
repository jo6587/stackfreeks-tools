How to Use 사용 방법
STEP 1

Type or paste Markdown

Write in the left editor — the preview on the right updates instantly on every keystroke.

마크다운 입력

왼쪽 에디터에 작성하세요. 오른쪽 미리보기가 키 입력마다 즉시 업데이트됩니다.

STEP 2

Use the toolbar

Click toolbar buttons to quickly insert bold, italic, headings, links, lists, and more.

툴바 활용

툴바 버튼을 클릭해 굵게, 기울임, 제목, 링크, 목록 등 서식을 빠르게 삽입하세요.

STEP 3

Copy the HTML output

Click "Copy HTML" to copy the rendered HTML to your clipboard — ready to paste anywhere.

HTML 복사

"HTML 복사" 버튼을 클릭하면 렌더링된 HTML이 클립보드에 복사됩니다.

STEP 4

Clear and start fresh

Hit the Clear button to wipe the editor and preview so you can start a new document.

초기화

지우기 버튼을 누르면 에디터와 미리보기가 모두 초기화되어 새 문서를 작성할 수 있습니다.

FAQ 자주 묻는 질문
What Markdown features are supported? 어떤 마크다운 기능이 지원되나요?
GFM (GitHub Flavored Markdown) — headings, bold, italic, lists, links, images, tables, code blocks, blockquotes, and horizontal rules. GFM(GitHub Flavored Markdown)을 지원합니다 — 제목, 굵게, 기울임, 목록, 링크, 이미지, 표, 코드 블록, 인용구, 수평선 등이 포함됩니다.
Does it support syntax highlighting? 코드 구문 강조(Syntax Highlighting)를 지원하나요?
Not currently — code blocks render clearly but without language-specific color highlighting. 현재는 지원하지 않습니다. 코드 블록은 렌더링되지만 언어별 색상 강조는 없습니다.
Can I paste from a file? 파일에서 붙여넣기 할 수 있나요?
Yes — open your .md file in any text editor, select all, and paste directly into the left editor panel. 네 — .md 파일을 텍스트 에디터에서 열고, 전체 선택 후 왼쪽 에디터에 붙여넣기 하면 됩니다.
Is my content saved? 작성한 내용이 저장되나요?
No — content lives in your browser only while the tab is open. Nothing is sent to a server. Copy your work before closing. 아니요 — 내용은 탭이 열려 있는 동안 브라우저에만 존재합니다. 서버로 전송되지 않으며, 닫기 전에 복사해 두세요.
