Frequently Asked Questions 자주 묻는 질문

HEIC (High Efficiency Image Container) is the default photo format used by iPhones since iOS 11. It uses the HEIF standard to store images at roughly half the file size of JPG while maintaining similar visual quality. Apple adopted it to save storage on devices, but the format is not universally supported outside the Apple ecosystem.
Windows and Android do not natively support the HEIC format. On Windows 10/11 you can install the optional "HEIC Image Extensions" codec from the Microsoft Store, but many apps still don't recognize the format. On Android, most gallery apps simply don't support HEIC. Converting HEIC to JPG is the most reliable way to ensure broad compatibility across all platforms and devices.
There is a small, often imperceptible quality loss when converting HEIC to JPG, because JPG uses lossy compression. At the default 92% quality setting in this tool, the output is visually nearly identical to the original HEIC. You can increase the quality slider to 100% for a lossless-looking output, though the file size will be larger. HEIC files converted at 92% quality typically produce JPGs that are 10–30% larger in file size than the original HEIC.
