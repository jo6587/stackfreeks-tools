# HEIC to JPG Converter

**Live tool:** [tools.stackfreeks.com/tools/heic-to-jpg/](https://tools.stackfreeks.com/tools/heic-to-jpg/)

Convert iPhone HEIC photos to JPG instantly in your browser — free, no account, no server upload.

## Features

- **100% browser-side** — files never leave your device
- **Batch conversion** — drop multiple HEIC files at once
- **Quality control** — adjustable JPG quality slider (70%–100%, default 92%)
- **Live preview** — thumbnail + before/after file size displayed per conversion
- **Download All** — one-click batch export when 2+ files converted
- **Bilingual UI** — English / Korean (EN/KO toggle)
- **Responsive** — works on desktop and mobile

## Tech stack

- Pure HTML / CSS / JS (no build step)
- [`heic2any`](https://github.com/alexcorvi/heic2any) v0.0.4 via CDN for HEIC decoding
- `/shared/i18n.js` for EN/KO language toggle
