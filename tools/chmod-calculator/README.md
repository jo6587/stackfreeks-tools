# Chmod Calculator

> Live: https://tools.stackfreeks.com/tools/chmod-calculator/

Interactive Linux file permission calculator. Convert between octal (755), symbolic (rwxr-xr-x), and chmod commands.

## Features
- Interactive 3x3 permission checkbox grid
- Two-way: edit octal or click checkboxes
- Common presets (644, 755, 600, 700, 777)
- Recursive -R flag toggle
- Special bits: setuid, setgid, sticky
- No account, no data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
