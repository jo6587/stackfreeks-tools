# Game Server Cost Calculator

> Live: https://tools.stackfreeks.com/tools/game-server-cost-calculator/

Estimate your game server VPS cost by player count and game type.

## Features

- Supports Minecraft, Valheim, Rust, ARK, CS2, Terraria, and custom games
- Live cost estimation based on concurrent players
- Vultr VPS plan recommendations with pricing
- EN/KO bilingual

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
