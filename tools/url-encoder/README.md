# URL Encoder / Decoder

> Live: https://tools.stackfreeks.com/tools/url-encoder/

Percent-encode URLs and query parameters, or decode encoded strings. Free, works in your browser.

## Features
- encodeURI / decodeURI (full URL)
- encodeURIComponent / decodeURIComponent (query params)
- Error handling for malformed URIs
- Percent-encoding cheat sheet
- No data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript (native browser APIs only)
- No build tools, no dependencies
- Deployed on Cloudflare Pages
