# Image Compressor

**URL:** https://tools.stackfreeks.com/tools/image-compressor/

A free, browser-based image compressor that supports JPG, PNG, and WebP files. No file size limit, no watermark, no server upload — all processing happens locally in the browser using the `browser-image-compression` library.

## Features

- **Batch support** — compress multiple images at once with no file count limit
- **Quality slider** — 10%–90%, default 80% (best balance of size and quality)
- **Max width options** — Original / 1920px / 1280px / 800px
- **Real-time progress** — per-file progress bar while compressing
- **Before/after stats** — file size comparison and savings percentage badge
- **Download All** — one-click batch download when 2+ files are done
- **Privacy-first** — zero server upload; files never leave the browser
- **No watermark, no sign-up, no limit**

## Tech Stack

- Pure static HTML/CSS/JS — no build step required
- [`browser-image-compression@2.0.2`](https://github.com/Donaldcwl/browser-image-compression) via jsDelivr CDN
- EN/KO bilingual via `/shared/i18n.js`
