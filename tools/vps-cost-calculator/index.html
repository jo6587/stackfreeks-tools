How to use this calculator

Step 1

Set your minimum RAM

Choose the minimum RAM your workload needs. A basic web server runs fine on 1–2 GB. Databases and app servers typically need 4 GB+.

Step 2

Set your minimum CPU

Select the minimum CPU core count. Most small projects only need 1–2 cores. CPU-intensive tasks like video processing or CI runners need 4+.

Step 3

Compare results

Plans are sorted by price. The cheapest matching plan gets the "Best value" badge. Check the storage column — it varies a lot between providers.

사용 방법

최소 RAM 설정

작업에 필요한 최소 RAM을 선택하세요. 기본 웹서버는 1–2 GB로도 충분합니다. 데이터베이스나 앱 서버는 보통 4 GB 이상이 필요합니다.

Frequently Asked Questions

A Virtual Private Server (VPS) is a virtualized server you rent from a cloud provider. Unlike shared hosting, you get dedicated CPU and RAM resources. VPS plans are ideal for running web apps, APIs, databases, game servers, or any always-on workload.
Hetzner operates primarily from data centers in Europe (Germany, Finland) and the US. Their lower operating costs and data center locations let them offer significantly lower prices. The trade-off: fewer regions and slightly longer latency for users in Asia or South America.
1 GB is enough for a lightweight static site or small Node.js/Python app. 2–4 GB covers most production web apps and small databases (MySQL, PostgreSQL). 8 GB+ is recommended for self-hosted tools, larger databases, or running multiple services on one server.

자주 묻는 질문

VPS(가상 사설 서버)는 클라우드 제공업체에서 임대하는 가상화된 서버입니다. 공유 호스팅과 달리 CPU와 RAM이 독립적으로 할당됩니다. 웹 앱, API, 데이터베이스, 게임 서버 등 상시 실행이 필요한 워크로드에 적합합니다.
Hetzner는 주로 유럽(독일, 핀란드)과 미국에 데이터센터를 운영합니다. 낮은 운영 비용 덕분에 훨씬 저렴한 가격을 제공할 수 있습니다. 단점은 리전이 적고, 아시아·남미 사용자는 레이턴시가 다소 높을 수 있습니다.
경량 정적 사이트나 소규모 Node.js/Python 앱은 1 GB면 충분합니다. 대부분의 프로덕션 웹 앱과 소규모 데이터베이스(MySQL, PostgreSQL)는 2–4 GB가 적당합니다. 셀프 호스팅 툴, 대형 DB, 여러 서비스를 한 서버에서 운영한다면 8 GB 이상을 권장합니다.
네, 목록의 모든 플랜은 NVMe/SSD 로컬 스토리지가 기본 포함됩니다. 단, 추가 블록 스토리지나 오브젝트 스토리지(S3 등)가 필요하면 별도 요금이 부과됩니다.
가격은 2025년 중반 기준으로 수동 관리되는 참고 데이터입니다. 클라우드 제공업체는 가격을 수시로 변경할 수 있으므로, 구매 전 반드시 해당 제공업체의 공식 가격 페이지에서 확인하세요.
