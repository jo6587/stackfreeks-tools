# VPS Cost Calculator

> Live: https://tools.stackfreeks.com/vps-cost-calculator/

Compare monthly VPS costs across Vultr, Hetzner, and DigitalOcean. Filter by RAM and CPU requirements to find the best-value plan instantly.

## Features

- Filter plans by minimum RAM and CPU cores
- Compares Vultr, Hetzner, and DigitalOcean
- Highlights the cheapest matching plan
- English / Korean bilingual UI
- Vultr affiliate CTA for monetization

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
