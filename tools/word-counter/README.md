# Word Counter

> Live: https://tools.stackfreeks.com/tools/word-counter/

Free online word counter — words, characters, sentences, reading time, keyword density.

## Features
- Live word/character/sentence/paragraph count
- Reading time estimate (200 wpm)
- Top 10 keyword density
- No account, no data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- No build tools, no dependencies
- Deployed on Cloudflare Pages
