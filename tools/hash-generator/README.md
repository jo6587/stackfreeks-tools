# Hash Generator

> Live: https://tools.stackfreeks.com/tools/hash-generator/

Generate MD5, SHA-1, SHA-256, and SHA-512 hashes from text or files. Free, works in your browser.

## Features
- MD5, SHA-1, SHA-256, SHA-512 — all shown simultaneously
- Text input (live) and file input (drag-and-drop)
- Uppercase/lowercase toggle
- No data sent to server

## Tech Stack
- Pure HTML/CSS/JavaScript
- MD5: spark-md5 (CDN)
- SHA-*: Web Crypto API (crypto.subtle)
- Deployed on Cloudflare Pages
